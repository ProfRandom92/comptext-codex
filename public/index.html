# Traditional Prompt (215 tokens)

Focus on code quality, potential bugs, security issues, and performance.
Provide specific suggestions for improvement with code examples.
Format your response as a structured report with sections."

# CompText Codex (65 tokens)

@format(report) @include(examples)
